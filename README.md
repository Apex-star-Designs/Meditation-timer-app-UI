# Meditation Timer App UI  

## Overview  
A calming, minimal mobile app interface for guided meditation.  
Includes timer controls, pause/reset functions, and a progress ring to enhance user focus.  

## Features  
- Clean and minimal interface  
- Timer start/pause/reset controls  
- Progress ring for visualization  
- Calming color palette and typography  

## Tools Used  
- Figma
